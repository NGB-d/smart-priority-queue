# 🚀 SmartQueue Pro - Priority Management System

A Full-Stack queue management application designed to optimize customer flow using priority-based sorting algorithms.

## 🧠 Core Features & Algorithms
- **Smart Priority Sorting:** Implemented a custom sorting logic ($O(n \log n)$) that reorganizes the queue based on urgency levels (Urgent, Medium, Regular).
- **Dynamic Waiting Time:** Real-time estimation algorithm that calculates wait times based on queue position.
- **Data Persistence:** Integrated Node.js File System (FS) to ensure data is saved permanently in a JSON database.
- **Modern UI/UX:** Responsive React-based interface with real-time updates and search capabilities.

## 🛠️ Technology Stack
- **Frontend:** React.js, CSS3 (Modern Flexbox/Grid), FontAwesome Icons.
- **Backend:** Node.js (Vanilla HTTP Server).
- **Data Storage:** JSON Persistence.

## 🏁 How to Run
1. Clone the repository.
2. Run `node server.js` in your terminal.
3. Open `index.html` in any browser.
